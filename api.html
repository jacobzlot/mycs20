Dog CEO API

This API is a massive collection of dog pictures that can be displayed upon request. I selected it because I love dogs and wanted to work with an API that is on the sillier side.

Source: https://dog.ceo/dog-api/

Parameters Used: This example uses the "breed" parameter in the URL (https://dog.ceo/api/breed/australian/images/random) to request a random image of a specific dog breed (australian). I chose australian because that is my dog's breed.

Example Use Case: While this is a silly API, it could definitely still be useful for dog adoption sites or sites that have educational content about dogs.
